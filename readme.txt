== Description ==

The **Creattica Items** plugin retrieves items from Creattica.com with your chosen RSS feed, then shows the results as linked images.  

== Installation ==

1. Upload the `creattica-items` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the built in widget to show your chosen RSS feed.
